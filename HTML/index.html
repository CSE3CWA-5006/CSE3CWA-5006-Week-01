Overflow

Scroll content when space is limited

CSS overflow controls what happens when content is bigger than the box. This can clip content, show scrollbars, or let content remain visible. Students often meet this when working with long text, fixed-height cards, chat panels, tables, and code blocks. Keep changing max-height, overflow, and white-space to see how the result changes in real time.

This CSS is often used in panels, cards, message areas, and long code blocks.
