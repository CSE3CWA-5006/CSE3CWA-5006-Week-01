# Welcome to CSE3CWA / CSE5006

Welcome to the **CSE3CWA / CSE5006** course repository.

In this course, you will learn practical skills for building, managing, and deploying cloud-based web applications. We hope you will gain useful knowledge, hands-on experience, and confidence in using modern development tools.

Before we begin the main course activities, please complete the following preparation tasks.

---

## 1. GitHub Account and Assessment Repository Setup

Please read this guide first:

[GitHub Account Registration and Assessment Repository Setup Guide](./README_GITHUB.md)

This guide explains how to:

* create or use a GitHub account
* choose a suitable GitHub username
* create a private assessment repository
* add the instructor as a collaborator
* submit your GitHub information in LMS

This step is required for assessment submission.

---

## 2. Ubuntu Virtual Machine Setup Using VMware

After completing the GitHub setup, please read this guide:

[Ubuntu Virtual Machine Setup Guide Using VMware](./README_VMWARE.md)

This guide explains how to:

* install VMware Workstation Pro / Player on Windows
* install VMware Fusion on MacBook
* choose the correct Ubuntu ISO file
* create an Ubuntu virtual machine
* install basic development tools
* check copy and paste, VMware Tools, and internet connection

This step will help you prepare a suitable Ubuntu environment for labs and development work.

---

## Before Class

Before the first practical session, please make sure that:

* you have access to your GitHub account
* your GitHub username is ready
* you understand how to create a private assessment repository and send invitation 
* you have installed or prepared your Ubuntu environment

If you experience technical issues, please contact your lecturer Dr Shuo Ding (s.ding@latrobe.edu.au) as early as possible.

We look forward to working with you throughout the semester.
